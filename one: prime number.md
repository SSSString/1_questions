# عدد اول

.برنامه ای بنویسید که عددی را از ورودی
گرفته، و بگوید آن عدد اول است یا نه
- را چاپ کند "is prime" در صورتی که آن عدد اول بود کلمه
- را چاپ کند "not prime" در غیر این صورت کلمه


# مثال
ورودی :
-     13
خروجی : 
-     is prime
# 
ورودی :
-     28
خروجی :
-     not prime


منبع سوال: مکتب خونه
